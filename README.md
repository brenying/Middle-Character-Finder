# Middle Character Finder

## Objective
This project was designed to practice string manipulation, indexing, and formatted console output by allowing users to input any line of text and receive the middle character or characters visually highlighted beneath the string.

### Skills Learned
- String indexing and substring extraction
- Loop construction and input validation
- Console formatting with printf
- Conditional logic for even/odd length handling
- Method creation for reusable features (title screen, clear screen)

### Tools Used
- Java 8+
- Scanner for input handling
- Console-based formatting and output
- Basic arithmetic logic for index calculations
